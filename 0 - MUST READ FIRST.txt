To run the desktop application on your computer, please follow these steps:
1) Download and install the latest version of Python for your system from https://www.python.org/downloads/
   Be sure to select the 'Add Python to PATH' option at the beginning of installation.
2) Navigate to the project folder using File Explorer, type "cmd" in the address bar at the top of the window, and press Enter. 
   This will open Command Prompt in the project folder.
3) Install all required Python modules and packages by running the following command in Command Prompt:
   `pip install -r requirements.txt`
4) You can now launch the desktop app by typing the following command in Command Prompt:
   `python MAIN.py`
